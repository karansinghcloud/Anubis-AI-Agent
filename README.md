# 📌 Project Overview

Anubis AI Agent is a Business and Data Analytics Assistant designed to automate repetitive analytical tasks, generate business insights, and assist in decision-making.

The goal of this project is to combine AI, Data Analytics, SQL, and Automation into a single intelligent workflow.

---

# 🎯 Vision

Build an AI Agent that can work like a Business Analyst and Data Analyst by helping users with:

- Data Analysis
- SQL Query Generation
- Dashboard Planning
- KPI Tracking
- Business Reporting
- Dataset Design
- Data Cleaning
- Resume & Portfolio Assistance
- Interview Preparation
